# Run Project
1. Clone repository
``` 
git clone 'link'
```
2. Install dependencies 
```
pnpm i
```
3. Clone file ```.env.template``` and rename the copy to ```.env```
4. Fill data into environment variables `(.env)`
5. Up database
```
docker compose up -d
```
6. Run application
```
pnpm start:dev
```
